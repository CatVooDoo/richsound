Music Streaming Platform — Product Description
A web-based music streaming application with two user roles: Listener and Author. Authors can also listen to music like regular listeners.

Authentication

Registration with role selection (Listener / Author)
Login / Logout
Session-based authentication


Author Features

Upload tracks (MP3, title, cover image)
Create and manage albums (group tracks, set cover, release date)
Create playlists
Listen to any music on the platform
View personal statistics dashboard:

Total plays per track
Total likes per track
Subscriber count
Plays chart by date




Listener Features

Browse and search tracks, albums, authors
Stream audio (play/pause, seek, volume)
Like tracks
Add tracks to favorites
Subscribe to authors
Create and manage personal playlists
Add/remove tracks from playlists


Music Player (global, bottom bar)

Persistent player across all pages
Controls: play/pause, previous/next, seek bar, volume
Displays: current track cover, title, author name


General Pages

Home feed: recently added tracks, popular tracks, new albums
Author public profile: bio, tracks, albums, subscriber count
Track page: cover, title, author, play button, like button
Album page: tracklist, cover, release date
Search: tracks, albums, authors


Technical Notes

Pure PHP with minimal OOP structure (Router, Database, Controllers, Views)
PDO for database access
Audio streaming via PHP with proper headers
File uploads stored in /public/uploads/
MySQL database
